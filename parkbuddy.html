Park Buddy

Project Link (Arduino): Github Repo (Arduino)

Project Demo: YouTube Demo

Park Buddy is a park assist system that helps drivers safely park their vehicles, especially while parking in reverse. It measures the distance between the parking car and the sensor by bouncing the ultrasonic/electromagnetic waves it generates and then emits an alarm tone at a predetermined speed in synchronous with the blinking LED, as well as displaying the computed distance on the Liquid Crystal Display.

Hardware components:

  • Arduino Uno
  • Ultrasonic Sensor
  • 3 LEDs
  • Buzzer
  • Liquid Crystal Display
  • Light Dependent Resistor
  • (3) 220Î© resistors
  • Temperature and Humidity Sensor
  • (2) Tactile switches
