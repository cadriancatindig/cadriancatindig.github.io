Node Package Deployment

The node package deployment project using TypeScript was assigned to me by our tech leads with the aim of enhancing and honing my skills in open-source code contribution. This project serves as an initial step towards publishing libraries or packages for the company, enabling developers and operations teams to work more efficiently. To achieve this, I utilized API calls with axios to gather third-party application data from Jenkins.

IThe primary purpose of this script is to retrieve third-party data via an API call, specifically the Database Healthcheck of the US Production systems. The data is then packaged, making it available for developers to utilize during migration to GitHub actions. By doing so, we can facilitate smoother and more effective development and operations processes within the company.
